# MBTI 测试网站设计方案

## 一、核心理念

### 1.1 基于荣格八维认知功能
不直接测试 E/I、S/N、T/F、J/P 四个维度，而是测试八个认知功能的强度：

**感知功能（Perceiving）：**
- **Ne (外向直觉)**：看到可能性、联想、头脑风暴
- **Ni (内向直觉)**：洞察本质、预见未来、抽象思考
- **Se (外向感觉)**：关注当下、实际体验、行动力
- **Si (内向感觉)**：回忆细节、经验总结、稳定性

**判断功能（Judging）：**
- **Te (外向思考)**：逻辑效率、系统化、目标导向
- **Ti (内向思考)**：内在逻辑、分析原理、精确性
- **Fe (外向情感)**：群体和谐、他人情绪、社交规则
- **Fi (内向情感)**：个人价值、真实感受、道德判断

### 1.2 16 种 MBTI 类型的认知功能栈

每个类型有 4 个主要功能（按优先级）：

| MBTI | 主导功能 | 辅助功能 | 第三功能 | 劣势功能 |
|------|---------|---------|---------|---------|
| INTJ | Ni | Te | Fi | Se |
| INTP | Ti | Ne | Si | Fe |
| ENTJ | Te | Ni | Se | Fi |
| ENTP | Ne | Ti | Fe | Si |
| INFJ | Ni | Fe | Ti | Se |
| INFP | Fi | Ne | Si | Te |
| ENFJ | Fe | Ni | Se | Ti |
| ENFP | Ne | Fi | Te | Si |
| ISTJ | Si | Te | Fi | Ne |

| ESFJ | Fe | Si | Ne | Ti |
| ISTP | Ti | Se | Ni | Fe |
| ISFP | Fi | Se | Ni | Te |
| ESTP | Se | Ti | Fe | Ni |
| ESFP | Se | Fi | Te | Ni |

## 二、问题设计原则

### 2.1 场景化而非直接提问

**❌ 传统方式：**
"你更喜欢独处还是和朋友在一起？"
→ 太直白，容易被猜到测的是 E/I

**✅ 场景化方式：**
"周五晚上，你刚结束一周的工作：
A. 想约朋友出去吃饭聊天，分享这周的趣事
B. 想一个人在家，点个外卖看看剧
C. 看情况，如果有有趣的活动就出去，没有就宅家"

### 2.2 测试认知功能而非表面行为

**测试 Ne (外向直觉) 的例子：**
"你在咖啡馆等朋友，看到隔壁桌一对情侣在争吵：
A. 好奇他们为什么吵架，脑补各种可能的剧情
B. 觉得不关我的事，继续看手机
C. 感到不舒服，希望他们小声点"

**测试 Ti (内向思考) 的例子：**
"朋友向你推荐一个很火的产品，你会：
A. 先研究它的工作原理，看是否符合逻辑
B. 看看大家的评价，如果口碑好就试试
C. 相信朋友的推荐，直接买"

### 2.3 放松的语言风格

- 用"你"而非"您"，更亲切
- 场景描述生动、有画面感
- 选项不要太长，3-4 个为宜
- 避免"总是/从不"这种绝对化表述

## 三、评分机制

### 3.1 每个问题对应认知功能

每个问题的选项对应不同的认知功能，例如：

```json
{
  "question": "你在咖啡馆等朋友...",
  "options": [
    {
      "text": "好奇他们为什么吵架，脑补各种可能的剧情",
      "scores": { "Ne": 3, "Fe": 1 }
    },
    {
      "text": "觉得不关我的事，继续看手机",
      "scores": { "Ti": 2, "Si": 1 }
    },
    {
      "text": "感到不舒服，希望他们小声点",
      "scores": { "Fi": 2, "Si": 1 }
    }
  ]
}
```

### 3.2 计算最终类型

1. **累加各认知功能得分**
2. **找出最强的感知功能**（Ne/Ni/Se/Si）
3. **找出最强的判断功能**（Te/Ti/Fe/Fi）
4. **根据功能栈匹配 MBTI 类型**

## 四、用户体验设计

### 4.1 测试流程

1. **欢迎页面** - 简单介绍测试理念
2. **测试页面** - 场景化问题，带进度条
3. **结果页面** - 显示类型和认知功能分析

### 4.2 视觉设计

- 柔和配色，不刺眼
- 每个场景配简单插图
- 平滑的页面过渡动画
- 响应式设计
