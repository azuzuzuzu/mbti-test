# 辩论式 MBTI 测试设计

## 核心理念
通过辩论的方式测试认知功能，在对抗中展现真实的思维模式。

## 辩题设计

### 辩题 1：效率 vs 公平
**辩题**："在资源有限的情况下，效率比公平更重要"

**测试目标**：
- 正方（效率优先）→ Te 倾向
- 反方（公平优先）→ Ti/Fi 倾向
- 论证方式：
  - 用数据和结果 → Te
  - 用逻辑推理 → Ti
  - 用价值判断 → Fi
  - 用群体影响 → Fe

### 辩题 2：传统 vs 创新
**辩题**："应该保留传统制度而非冒险创新"

**测试目标**：
- 正方（保留传统）→ Si 倾向
- 反方（大胆创新）→ Ne 倾向
- 论证方式：
  - 引用历史经验 → Si
  - 提出多种可能 → Ne
  - 预见长远影响 → Ni
  - 关注当下问题 → Se

### 辩题 3：个人 vs 集体
**辩题**："个人自由比集体利益更重要"

**测试目标**：
- 正方（个人自由）→ Fi 倾向
- 反方（集体利益）→ Fe 倾向
- 论证方式：
  - 强调个人价值 → Fi
  - 强调群体和谐 → Fe
  - 分析权衡 → Ti/Te

### 辩题 4：理想 vs 现实
**辩题**："追求理想比面对现实更重要"

**测试目标**：
- 正方（追求理想）→ Ni/Ne 倾向
- 反方（面对现实）→ Si/Se 倾向
- 论证方式：
  - 描绘愿景 → Ni
  - 想象可能 → Ne
  - 引用事实 → Si
  - 强调行动 → Se

## 辩论流程

1. **选择立场**：用户选择正方或反方
2. **开场陈述**：用户阐述自己的观点（200字内）
3. **AI 反驳**：我针对用户的论点进行反驳
4. **用户反驳**：用户回应我的反驳
5. **深入交锋**：2-3 轮对话
6. **总结陈词**：用户做最后陈述

## 评分维度

### 论证方式
- **逻辑结构**（Ti）：论证是否严密、自洽
- **实用效果**（Te）：是否关注结果和效率
- **价值判断**（Fi）：是否基于个人信念
- **群体考量**（Fe）：是否考虑他人感受

### 信息处理
- **经验引用**（Si）：是否引用历史、经验
- **当下关注**（Se）：是否关注具体事实
- **深度洞察**（Ni）：是否看到本质和趋势
- **多元思考**（Ne）：是否提出多种可能

### 表达风格
- **直接果断** → Te/Se
- **谨慎细致** → Si/Ti
- **热情感染** → Fe/Ne
- **坚定深沉** → Fi/Ni

## 技术实现

### 前端
- 辩题选择页面
- 立场选择
- 对话式界面（类似聊天）
- 实时显示论证分析

### 后端
- 需要 AI 接口来生成反驳
- 或者预设多种反驳模板
- 分析用户输入的关键词和结构

### 简化版本（无 AI）
- 预设 3-4 轮固定的反驳
- 用户选择回应方式（多选项）
- 根据选择累计得分
